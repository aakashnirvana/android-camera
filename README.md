# 📷 Android Camera App

This repository contains a **simple Android Camera application** developed in **Java**, showcasing how to use the **Android Camera API** to capture photos and videos. It includes a basic preview interface and serves as a minimal example for beginners or as a foundation for more advanced camera apps.

---

## 🎯 Features

- Access and launch the **device's camera**
- Capture **images** and/or **videos**
- View a **basic camera preview**
- Straightforward, lightweight, and **ideal for learning or quick prototyping**

---

## 🛠️ Technologies Used

- Java
- Android SDK (Camera API)
- Android Studio (Recommended IDE)

---

## 🚀 Getting Started

### ✅ Prerequisites

- **Android Studio** installed on your system.
- An **Android device** or emulator with camera support.
- Minimum SDK version as specified in the project (see `build.gradle`).

---

## 🔧 How to Run

1. **Clone the Repository**

```bash
git clone https://github.com/aakashnirvana/android-camera.git
cd android-camera
Open in Android Studio

Launch Android Studio.

Click "Open an existing project".

Select the cloned folder (android-camera/).

Build and Run the App

Connect an Android device via USB (enable developer mode) or launch an emulator.

Click Run ▶️ or press Shift + F10.

The app will launch and allow you to use the camera.
